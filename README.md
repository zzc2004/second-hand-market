# 二手交易平台

基于Java的校园二手商品交易平台，提供商品发布、在线交易等功能。

## 声明

仅用于个人课程设计代码存储及学习交流

## 主要功能
- 用户注册/登录
- 商品信息发布与管理
- 商品搜索与分类浏览
- 个人信息管理
- 购物车管理
- 订单管理

## 技术栈
### 后端
- Java 21
- Spring + SpringMVC + MyBatis
- MySQL 8.0

### 前端
- JSP
- Bootstrap
- JavaScript
- CSS

### 环境要求
- JDK 21
- Maven 3.9.8
- MySQL 8.0.33
- Tomcat 10.1.18
